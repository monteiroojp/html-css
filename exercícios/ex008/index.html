Principais formatações

Textos em negrito/destaque

Para colocar algo em negrito bastar usar a tag < strong >
Ex: Trypanossoma cruzi

Textos em itálico

Para colocar algo em itálico bastar usar a tag < em >
Ex: Trypanossoma cruzi

Textos marcados

Para colocar um texto marcado basta usar a tag < mark >
Ex: Lions cia

Textos com tamanhos diferentes

Para fazer um texto menor dentro de um mesmo paragrafo basta usar a tag < small >
Ex: Texto menor
obs: Existe uma tag para textos grandes, a < big > , mas ela é obsoleta pois não traz sentido consigo

Texto deletado

Para fazer uso de um texto deletado(riscado), basta usar a tag < del >
Ex: Texto riscado

Texto inserido(sublinhado)

Para colocar um texto sublinhado basta usar a tag < ins >
Ex: Texto sublinhado

Texto sobrescrito

Para colocar um texto sobrescrito, basta usar a tag < sup >
Ex: x2 +10x +9 = 0

Texto subscrito

Para colocar um texto subescrito, basta usar a tag < sub >
Ex: MgO4
