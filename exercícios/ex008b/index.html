Demais formatações restantes

Fonte de código(monoespaçada)

Para digitar na forma de código bastar usar a tag < code >
Ex: document.getElementById

Tabulação de códigos

Para conseguir escrever códigos da forma correta basta usar a tag < pre >
Ex: 

    num_1 = int(input("Digite um número: "))
    num_2 = int(input("Digite outro número: "))
    if num_1 > num_2:
        print("O primeiro número é maior")
    elif num_1 < num_2:
        print("O segundo número é maior")
    else:
        print("Os números possuem o messmo valor")

Citações

Para fazer uma citação normal, basta usar a tag < q >
Ex: De acordo com Thomas Hobbes, o governo é reponsável por garantir o bem-estar do cidadão

Também tem como fazer citação em bloco usando a tag < blockquote >
ex: De acordo com Thomas Hobbes,

o estado é responsável por garantir o bém-estar do cidadão

Siglas

É possivel informar ao usuário qual o singificado de uma sigla por meio do uso da tag < abbr >
Ex: Participo da MW e estou estudando HTML

Também é possível escrever textos invertidos usando a tag < bdo >
Ex: A irmã do kraken é uma safada
