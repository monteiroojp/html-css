Tipos de listas

Listas ordenadas

Para designar o uso de uma lista ordenada se usa a tag < ol > , enquanto para designar um item < li >
ex:

  1. Acordar
  2. Entrar no discord
  3. Falar Vai tomar no cú caraio

Também pode ser alterado os parâmetros dessa lista
Ex:

  1. Acordar
  2. Entrar no discord
  3. Falar Vai tomar no cú caraio

Listas não ordenadas

Também é possivel fazer listas não ordenadas usando a tag < ul >


Ex:
  • Acordar
  • Entrar no discord
  • Falar Vai tomar no cú caraio

Da mesma forma que nas listas ordenadas, também pode alterar o elemento que acompanha os itens da lista com o parâmetro "type"


Ex:
  • Acordar
  • Entrar no discord
  • Falar Vai tomar no cú caraio

Listas mistas pt.1

Também é possivel criar listas ordenadas misturadas com as não ordenadas
ex:

  1. Filmes discord pandemia
    • Filme de netal
    • Interestelar
    • Invocação do mal
  2. Filmes disord pós pandemia
    • Inatividade paranormal 1
    • Inatividade paranormal 2
    • Inatividade paranormal reprise

Listas mistas pt.2

Também é possiveel criar listas ordenadas dentro de outra lista ordenada. Além disso o mesmo é possivel com listas não ordenadas dentro de outras listas não ordenadas.
Ex:

  1. Filmes discord pandemia
    1. Filme de natal
    2. Interestelar
    3. Invocação do mal
  2. Filmes discord pós pandemia
    1. Inatividade paranormal 1
    2. Inatividade paranormal 2
    3. Inatividade paranormal reprises
