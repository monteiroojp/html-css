Links internos pt.1

Também é possível, como nas imagens, colocar links internos, desde que o arquivo .html esteja presente na mesma pasta

Links internos pt.2

Para fazer com que o link interno abra em outra aba, é necessário o uso dos parâmetros target:_blank
Ex: Página 02

Links duvidosos

Para links duvidosos, ou seja, que não deseja que o mecanismo de busca associe eles ao seu site, basta usar o parametro hell=nofollow
Ex: Página 02

Links dentro de pastas

Links em pasta diferentes não aparecem como opção direta, por isso é preciso colocar seu diretório sempre.
Obs: Caso esteja numa pasta anterior, use ../ antes do nome do arquivo
Ex: Página 03
