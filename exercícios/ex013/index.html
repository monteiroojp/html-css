Estilos inline

Nesse estilo basta usar o parâmetro "style" para alterar a forma do jeito que desejar, porém nesse método é necessário fazer muitas das vezes algo muito manual e trabalhoso.

Utilidade real

A utilidadae real desse estilo de CSS está na realização de mudanças pontuais, porém nunca para realização de projetos em grande esacala.
